My Practice Page for Class #4

Begin: set it up, edit it, and push it. Link to the page of someone else in the class.

  2. write out a description, and build a basic web page.
  3. Gather at least 4 images, put them in
    tags with some text. Use css to size your images.
  4. Include an image stored on someone else's server.
  5. Link to someone else's page.
  6. Do something interesting with those images. Perhaps create a multi-page portfolio like Isabella's, use thumbnails and pop-up images, or use a slider (download and integrate one).
