Learning & Practice Goals (begin):

Templates and images are here or just grab your own.

  • Create html documents, with text, headers, links and images. Sunflower Example. Or Van Gogh without image, and then add an image. Create two html docs and link them together. At some point, link to your pair-programming partner's page.
  • Add images. Add at least 4. Include some on your own server, and link to your pair-programming partner.
  • Basic css styling.
    1. Choose your image size
    2. Change something stylistic (font, color)
    3. Change something format-oriented (create columns with floats. Use padding and margins)

Get as far as you can get. If you get stuck, just copy an example and keep going. If you find it too easy, make it look good, work on UI/UX!

Class #3: Proficiency with Everything Basic Before Coding

Start class by getting GitHub and your text editor (Sublime?) set up. Then download some sample html html exercise #1 and html exercise #2, and practice during lectures.

Technology Goals:

Networking: what is the internet, how to get a web page. Proficiency with GitHub, understanding basics of how the web is wired together.
Chrome tools: edit in place to experiment

Learning & Practice Goals:

Basic website creation -- start your project! Gather photos, start structuring them, decide how many pages you want, create stub pages and link them. Preferred goals: (a) display images at size and location of your choice, (b) link to your pair-programming partner's website.

The intention is to create a page with small photos that expand (using scripts in the next class) and/or that are in a slider.
