See basic examples of jquery sliders and other toys and a very simple bootstrap example

Grids join floats and relative positioning, as ways to move content around your page. They are not fundamental to css: they are built with css and javascript, for you to use. jQuery is a javascript library full of shortcuts. Bootstrap is a set of css and javascript that let you create a themed website without doing all the work yourself.

Today is mostly for working on your class projects. Get Sublime, load/pull/clone/[keep using] the most recent commit of your project, and experiment with grids and javascript tools that have been built for you.
